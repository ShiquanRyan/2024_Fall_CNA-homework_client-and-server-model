
# 2024Fall_Computer Network and Application in NTUIM
台大資管系大三必修網路技術與應用，此次作業的範圍為利用 socket programming 概念製作簡易的 client and server 模型和 P2P 模型。



## Description
- 題目名稱：第三方支付使用者對使用者小額付款系統
- 作業一： client 端實作，功能包含：
    - 用戶註冊
    - 用戶登入
    - 用戶確認餘額與系統上線清單
    - 用戶退出
    - 用戶轉帳給其他用戶
- 作業二：server 端實作，功能包含：
    - 辦理用戶註冊
    - 協助用戶登入系統
    - 回傳餘額以及系統上線清單
    - 協助用戶退出
    - 更改轉帳金額
